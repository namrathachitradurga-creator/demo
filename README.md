## How to Run

1. Clone the repository using HTTPS URL

git clone https://github.com/namrathachitradurga-creator/demo.git

2. Change directory to the project folder

cd demo

3. Create virtual environment

python -m venv .venv

4. Activate virtual environment (Windows)

.venv\Scripts\Activate.ps1

5. Install Flask

pip install flask

6. Navigate to React application folder

cd my-dynamic-app

7. Start the React development server

npm start

8. Open browser

http://localhost:3000
